## Security Policy

### Reporting Security Issues

If you discover a security vulnerability, please email security@taskmaster.ai instead of using the issue tracker.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested fix (if any)

### Security Best Practices

When using TaskMaster AI:
- Keep your API keys secret and never commit them to the repository
- Use environment variables for sensitive configuration
- Keep dependencies updated
- Use HTTPS in production
- Validate all user inputs on the backend
